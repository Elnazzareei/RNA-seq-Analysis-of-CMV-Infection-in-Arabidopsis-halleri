# Objective
The objective of this project is to analyze the high-dimensional gene expression data to identify patterns and differences across experimental conditions.

# Dataset
-Public RNA-seq dataset from GEO (GSE197739)
-12 samples across 4 experimental conditions
-~33,000 gene expression features

# Tools & Technologies
-Python (Pandas, NumPy)
-Statistical analysis (SciPy, Statsmodels)
-Visualization (Matplotlib, Seaborn)
-Machine learning (Scikit-learn)

# Workflow
-Data acquisition and integration from public database
-Data cleaning and preprocessing (merging, formatting, normalization)
-Exploratory Data Analysis (EDA)
-Dimensionality reduction using PCA
-Differential expression analysis
-Data visualization (PCA plots, heatmaps, volcano plots)

# Key Results
-Identified significantly different features across experimental conditions
-Revealed clear separation patterns using PCA
-Highlighted key variables contributing to observed differences

# Key Insights
-High-dimensional data can be effectively reduced to identify meaningful patterns
-Statistical analysis enables detection of significant changes across conditions
-Visualization techniques improve interpretability of complex datasets
